# 포토스티커

## 사전지식

### `GestureDetector`와 `제스쳐`

- 플러터에서 지원하는 모든 제스쳐를 구현할 수 있는 `Widget`
- 제스쳐가 인식 되면 `매개변수`에 입력된 `콜백`이 실행 됨
- 많이 사용하는 제스쳐들
    - onTap
    - onDoubleTap
    - onLongPress
    - onScale
    - onVerticalDragStart
    - onVerticalDragEnd
    - onHorizontalDragStart
    - onHorizontalDragEnd
    - onPanStart
    - onPanEnd
